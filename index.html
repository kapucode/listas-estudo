Lista Ordenada

Para adicionar uma lista ordenada em seu código HTML, você deve adicionar a tag <ol>. Dentro dessa tag você deve adicionar a tag <li>, que adiciona um item de lista.

Passo a passo para criar uma lista ordenada:

  1. Abrir e fechar a tag <ol>
  2. Dentro dela, adicionar uma tag <li> a cada item da lista.

Atributos de <ol>

Todos: <ol type="1/A/a/I/i" start="">

type="1" - Deixa a lista ordenada com números.

type="A/a" - Deixa a lista ordenada com o alfabeto maiúsculo ou minúsculo.

type="I/i" - Deixa a lista ordenads com números romanos maiúsculos ou minúsculos.

start="" - De onde a lista irá começar (4, D, IX)

Lista Não Ordenada

Para adicionar uma lista não ordenada em seu código HTML, você deve adicionar a tag <ul>. Dentro dessa tag você deve adicionar a tag <li>, que adiciona um item de lista.

Passo a passo para criar uma lista não ordenada:

  • Abrir e fechar a tag <ul>
  • Dentro dela, adicionar uma tag <li> a cada item da lista.

Atributos de <ul>

Todos: <ul type="disc/circle/square">

type="disc" - Deixa o estilo da lista com uma bolinha preenchida..

type="circle" - Deixa o estilo da lista com uma bolinha não preenchida..

type="square" - Deixa o estilo da lista com um quadrado preenchido..

Juntando listas

Para juntar, é preciso somente aninhar as listas. <ol> pode ser junto dele mesmo e de <ul>, com <ul> é a mesma coisa.

Lista de definição

A tag <dl> é utilizada para fazer uma lista de definições de palavras. Como um: HTML
Linguagem de marcação utilizada para desenvolvimento web.

Passo a passo para criar uma lista de definição:

  1. Adicione a tag <dl> (Definition List)
  2. Dentro dele, adicione a tag <dt> (Definition Text)
  3. Após isso, adicione a tag <dd> (Definition Description)
